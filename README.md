# 🏥 Healthcare Operations Analytics

## 📌 Project Overview

This project focuses on analyzing hospital operational data using SQL to evaluate hospital performance, departmental efficiency, patient volume, doctor utilization, and medical expenses. Advanced SQL techniques were used to uncover meaningful insights and support data-driven decision-making in healthcare management.

---

## 🎯 Project Objectives

- Analyze hospital revenue and performance across multiple locations.

- Evaluate department-wise patient volume and medical expenses.

- Measure resource utilization using patient-to-doctor ratios.

- Identify high-performing hospitals and departments.

- Analyze patient stay durations and operational efficiency.

- Generate actionable business insights through SQL analytics.

---

## 🛠️ Tools & Technologies Used

- SQL (MySQL)

- Excel

---

## 📂 Dataset Information

The dataset contains healthcare operational records with the following attributes:

| Column Name | Description |

|------------|-------------|

| Hospital Name | Name of the hospital |

| Location | Hospital location |

| Department | Medical department |

| Doctors Count | Number of doctors available |

| Patients Count | Number of patients treated |

| Admission Date | Patient admission date |

| Discharge Date | Patient discharge date |

| Medical Expenses | Treatment expenses incurred |

## 🔍 Business Problems Solved

### Hospital Performance Analysis

- Ranked hospitals based on total revenue generated.

- Identified hospitals performing above average revenue.

- Calculated cost per patient across hospitals.

- Compared hospital revenue contribution.

### Department Analysis

- Identified the highest revenue-generating departments.

- Calculated department-wise revenue contribution percentages.

- Analyzed patient-to-doctor ratios by department.

- Ranked departments based on patient volume.

### Resource Utilization Analysis

- Evaluated doctor workload across hospitals.

- Identified hospitals with high patient-to-doctor ratios.

- Measured operational efficiency using utilization metrics.

### Financial Analysis

- Detected high-cost treatment records.

- Analyzed revenue trends and expense distribution.

- Compared hospital and department-level financial performance.

### Patient & Stay Analysis

- Calculated average patient stay duration.

- Identified departments with longer treatment periods.

- Evaluated patient volume trends across hospitals.

---

## 📈 Key Insights

- **Healing Touch** generated the highest medical revenue of **₹352,178**, followed by **City Hospital (₹347,988)** and **Fortis Care (₹337,677)**.

- **Urology** emerged as the highest revenue-generating department with **₹433,418** in total medical expenses.

- The dataset recorded an **average patient stay duration of 7.95 days**, indicating that most treatments required approximately one week of hospitalization.

- **Urology, Neurology, and Dermatology** collectively contributed over **₹1.06 million** in medical expenses, accounting for a significant share of total revenue.

- Revenue distribution varied substantially across hospitals, with the top-performing hospitals generating more than **2×** the revenue of several lower-performing facilities.
